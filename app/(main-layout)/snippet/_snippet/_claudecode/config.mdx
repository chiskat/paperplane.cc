title: settings.json
---

可在 Claude Code [官网文档](https://code.claude.com/docs/en/settings) 查阅 `settings.json` 完整配置项；
在 [环境变量文档](https://code.claude.com/docs/en/env-vars) 查阅 `ANTHROPIC_` 开头和其它开头的环境变量配置项。

给出一份推荐配置：

```json
{
  "$schema": "https://json.schemastore.org/claude-code-settings.json",
  "language": "chinese",
  "skipWebFetchPreflight": true,
  "autoConnectIde": true
}
```
