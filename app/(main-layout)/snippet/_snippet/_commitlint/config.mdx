---
title: .commitlintrc
---

最常用的 Commit Message 约定，配置文件 `.commitlintrc` 内容如下：

```json
{
  "extends": ["@commitlint/config-conventional"],
  "rules": {},
}
```

也可以使用其它的 CommitLint 配置，例如：

- `@commitlint/config-lerna-scopes`
- `@commitlint/config-angular`

使用这些配置前，记得先安装它们。

---

`commitlint` 的配置是 JSON 格式，可以把配置放在 `/.commitlintrc`、`/.commitlintrc.json` 文件中，或者放置于 `package.json` 中的 `commitlint` 字段中；
如果使用方式比较复杂，还可以使用 `/.commitlintrc.js` 通过代码来定制它的行为。
