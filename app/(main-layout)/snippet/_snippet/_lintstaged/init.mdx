---
title: 初始化
---

可在 GitHub [文档页](https://github.com/lint-staged/lint-staged) 了解 LintStaged 的用法。

安装 `lint-staged`：

```bash
npm add -D lint-staged
```

配置 `husky`：
可以配置 `pre-commit` hook，在提交前对文件进行检查；
在 [Husky / LintStaged](/snippet/husky/lintstaged) 章节查看如何配置。

完成以上配置后，可在 [LintStaged / `.lintstagedrc`](/snippet/lintstaged/config) 章节查看配置。
