title: 下载和安装 Windows
group: Windows
---

以下为微软官网的 Windows 镜像下载地址：

- [Windows 11](https://www.microsoft.com/zh-cn/software-download/windows11)
- [Windows 11 ARM64](https://www.microsoft.com/zh-cn/software-download/windows11arm64)
- [Windows 10](https://www.microsoft.com/zh-cn/software-download/windows10)

请在网页中操作，版本建议选择 “专业中文版”；按步骤选择完成后点击开始下载，最终下载完成是 `.iso` 后缀的镜像文件。

---

建议使用 [Rufus](https://rufus.ie/zh/) ([GitHub](https://github.com/pbatard/rufus)) 制作 Windows 安装 U 盘。此工具具备优秀的速度和功能。
