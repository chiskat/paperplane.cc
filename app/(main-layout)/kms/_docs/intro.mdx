---
title: PaperPlane KMS 激活服务
---

KMS 是微软官方提供给企业用于批量激活 Windows 和 Office 的服务，允许企业自搭激活服务器，甚至允许纯内网环境下激活。
而 KMS 的协议已经被解密和在 GitHub [`Wind4/vlmcsd`](https://github.com/wind4/vlmcsd) 公开，任何人都可以自行实现和部署。

KMS 激活只需要连接目标服务器的 `1688` 端口，并进行密钥验证，验证完成即可激活；这个过程没有安全风险，所以网上的公开 KMS 服务器都可以安全使用；但后续大约每 180 天，系统会重新验证，因此需要寻找尽可能稳定的 KMS 服务器。

PaperPlane.cc 基于 `Wind4/vlmcsd` 实现了 Docker 镜像，以提供 KMS 服务。
KMS 激活服务器地址：

```
kms.p01.cc
```

网上也有很多公开的 KMS 服务器，因为只需要部署一个 Docker 镜像并放通 `1688` 端口，成本很低。
例如：[公开 KMS 服务器列表 1](https://www.wujingquan.com/470.html)、[公开 KMS 服务器列表 2](https://monitor.yerong.org/kms/)

> 注意：网上有很多本地运行的 KMS 激活，需要下载 `.exe` 文件运行，这种激活方式有风险，因为 `.exe` 可能含有病毒，而且 KMS 激活器往往需要关闭杀毒软件。
> 而且，KMS 激活后每隔 180 天需要重新连接服务器验证一次，所以这些激活器会在电脑上安装持久运行的服务，用于后续的激活验证。
