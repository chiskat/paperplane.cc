基于 Next.js、tRPC、`@tanstack/react-query` 和 Prisma 的先进轻量 Node.js 全栈项目。

以下是简介：

- 基于 Next.js、Shadcn UI、Tailwindcss 构建前端界面；
- 使用 `motion` 设计了组件过渡动画；
- 使用 `@tanstack/react-form` 和 `zod` 实现数据校验和表单处理，后端有自行开发的 trpc 数据处理管线；
- 使用 `@next/mdx` 相关工具解析 Markdown 并渲染，自行开发了 remark 和 rehype 插件解析显示 Markdown 特殊格式内容；
- 使用 Prisma 和 tRPC 来实现后端数据增删改查；
- 使用 `@tanstack/react-query` 集成 tRPC，SSR 中所有数据尽可能以 Hydrate 形式供给前端，仅在变更时再发送请求，减少不必要的网络开销；
- 使用 `better-auth` 的 OAuth2 结合自部署的 Gitea 来实现登录，结合 tRPC 中间件实现权限控制；
- 自带配套的运行容器、构建命令、容器开发配置。
